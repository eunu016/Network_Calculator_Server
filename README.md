
# Java Network_Calculator_Server Wiki

## 1. 개요

이 프로젝트는 Java Socket API를 활용한 클라이언트-서버 기반의 계산기 입니다.

***

## 2. 애플리케이션 프로토콜 정의

메시지는 ASCII 코드 기반이며, 각 필드는 공백(`DELIMITER`)을 기준으로 구분됩니다.

### 2.1. 클라이언트 요청 형식

요청은 **`COMMAND OPERAND1 OPERAND2`** 형식입니다.

* **명령어:** `ADD`, `SUB`, `MUL`, `DIV`.
* **ex)** `DIV 100 25`.

### 2.2. 서버 응답 형식

서버 응답은 **`CODE TYPE VALUE`** 세 필드로 구성됩니다.

| 코드 (CODE) | 유형 (TYPE) | 설명 |
| :--- | :--- | :--- |
| **200** | `ANSWER` | 계산 성공 및 결과 반환. |
| **400** | `INVALID_ARGS` | 인수 개수나 타입 불일치 등 클라이언트 오류. |
| **400** | `DIVIDE_BY_ZERO` | 0으로 나누기 연산 시도. |
| **400** | `UNKNOWN_COMMAND` | 지원하지 않는 명령 수신. |

#### 응답 값(`VALUE`) 처리 규정:

클라이언트 오류 응답 시 `VALUE` 필드에 공백이 포함된 상세 메시지가 포함되기 때문에 클라이언트에서는 **`String.split(DELIMITER, 3)`** 을 사용하여 `CODE`와 `TYPE` 외의 나머지 문자열 전체를 `VALUE`로 추출합니다.

## 3. 사용 방법

### 3.1. 환경 설정

1.  프로젝트 루트에 **`server_info.dat`** 파일 생성.
2.  파일의 첫 줄에 서버 IP 주소 (예: `127.0.0.1`), 둘째 줄에 Port 번호 (예: `1234`)를 입력.

### 3.2. 프로그램 실행

1.  **서버 실행:** 터미널에서 `java ServerEx` 명령을 실행.
2.  **클라이언트 실행:** 별도의 터미널에서 `java ClientEx` 명령을 실행 후 커맨드를 입력하여 서버와 통신 (`QUIT` 입력 시 클라이언트 종료).
